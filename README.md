# projeto-bot-python

Um bot simples em Python de previsão do tempo para Telegram.

## Pré-requisitos
- Python 3.10 ou superior
- Telegram

## Instalação
1. Clone esse repositório e navegue para o diretório
```bash
git clone https://github.com/andre-dev-2021/projeto-bot-python
cd projeto-bot-python/projeto-bot-python
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

## Configuração
1. Crie um arquivo .env com as chaves de API:
- `TELEGRAM_TOKEN` — token do bot no Telegram.
- `WEATHER_TOKEN` — chave de API para Open Weather API

2. Crie e configure um bot no Telegram

3. Adicione os seguintes comandos

- /clima
- /previsao
- /help

## Como executar

1. Inicie o arquivo python
```bash
python main.py
```

2. Com o telegram procure pelo seu bot e digite algum dos comandos disponiveis.
